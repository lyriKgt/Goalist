# 🎯 Goals Bot — Telegram бот для целей

## Что умеет
- Ставить цели на **неделю / месяц / год**
- **1-го числа каждого месяца** присылает опрос по месячным целям (да/нет)
- **Раз в квартал** (1 января, апреля, июля, октября) — опрос по годовым целям
- Невыполненные цели **автоматически переносятся** на следующий период
- **Inline-режим** — вызов через `@имя_бота` в любом чате → выбрать период → поделиться целями

---

## Деплой на Railway (бесплатно, ~5 мин)

### Шаг 1: Создать бота в Telegram
1. Открыть [@BotFather](https://t.me/BotFather)
2. `/newbot` → задать имя и username
3. Скопировать **BOT_TOKEN**
4. Включить inline-режим: `/setinline` → выбрать бота → написать любой placeholder (напр. `Введи: месяц / год / неделя`)

### Шаг 2: Залить код на GitHub
1. Создать репозиторий на [github.com](https://github.com)
2. Загрузить все файлы из этой папки

### Шаг 3: Деплой на Railway
1. Зайти на [railway.app](https://railway.app) → войти через GitHub
2. **New Project → Deploy from GitHub repo** → выбрать репозиторий
3. В разделе **Variables** добавить:
   ```
   BOT_TOKEN = твой_токен_сюда
   ```
4. Нажать **Deploy** — Railway сам установит зависимости и запустит бота

### Готово! 🚀
Railway даёт $5 кредитов/месяц — лёгкий бот тратит ~$0.5-1/мес, т.е. бесплатно на несколько месяцев.
Для постоянного бесплатного хостинга добавь карту (не списывает если в лимите).

## Команды бота
| Команда | Действие |
|---------|---------|
| `/start` | Главное меню |
| `/menu` | Главное меню |
| `@бот месяц` | Inline: показать цели на месяц |
| `@бот год` | Inline: показать цели на год |
| `@бот неделя` | Inline: показать цели на неделю |
